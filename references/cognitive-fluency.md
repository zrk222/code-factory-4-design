# Cognitive Fluency — playbook
The brain reads "easy to process" as "trustworthy, higher quality."
- Hick's Law: fewer choices; primary nav ≤7; one goal per section.
- Miller's Law: chunk into 5–9 units via Grid/Flex `gap`.
- Extreme whitespace (`8vw`/`8rem`): "too confident to shout" = premium.
- Typography: line-height ≥1.6, clear size hierarchy, short blocks (Georgia/Verdana/modern sans).
- Fitts's Law: big primary CTA in a high-traffic zone.
- Figure/ground: z-index, box-shadow, backdrop-filter to pull CTAs/modals forward.
